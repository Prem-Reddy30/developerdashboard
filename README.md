# Developer Dashboard

This project is a Developer Dashboard built with React and Vite. It provides a visual overview of repository analytics and developer activity.

## Tech Stack
- **Frontend**: React, Vite
- **Backend**: Node.js/Express (in `backend/` directory)
- **Styling**: Vanilla CSS / Tailwind CSS

## Project Setup

### Frontend
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`

### Backend
1. Go to backend directory: `cd backend`
2. Install dependencies: `npm install`
3. Set up `.env` file with necessary API keys.
4. Run server: `node server.js`
